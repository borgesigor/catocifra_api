## Documentation

#### Get all users

```http
  GET /user/listall
```

| Parâmetro   | Tipo       | Descrição                           |
| :---------- | :--------- | :---------------------------------- |
| `api_key` | `string` | Your api key |
| `author_id` | `string` | The author, needs to have a 'add_permission' |
| `target_id` | `string` | The target |

Return: `USER`
